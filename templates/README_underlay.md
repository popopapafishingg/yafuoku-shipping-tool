# 佐川プレビュー用ひな形

## 同梱ファイル

- **`sagawa_form_scan.pdf`** … 未記入用紙のスキャン（位置調整プレビューの下地に使用）
- **`sagawa_underlay.png`** … 上記 PDF から生成した画像（PDF が無いときの予備）

## プレビューの見方

1. アプリで **送り状の位置調整** を開く。
2. **プレビューを開く** → ブラウザでは `http://127.0.0.1:…/preview.html` を使う（`file://` の PDF は更新されないことがあります）。
3. スキャン画像の上に **サンプル文字** と **薄い破線の枠** が重なります。枠とマスに合わせて数値を直し、**保存して閉じる**。

本番の印刷 PDF にはスキャン画像は入りません（プレビュー専用です）。

## スキャンを差し替える場合

新しいスキャン PDF を `sagawa_form_scan.pdf` としてこのフォルダに置き、
アプリを再起動してください。
